# Dart Complete Course

A beginner-friendly repository for learning the fundamentals of the Dart programming language through practical examples and exercises.

## 📚 Topics Covered

- Variables
- Data Types
- Input & Output
- Operators
- String Manipulation
- Type Casting
- Conditional Statements (`if`, `else if`, `switch`)
- Loops (`for`, `while`, `do while`)
- Functions
- Lists
- Maps
- Sets
- Classes & Objects
- Constructors
- Encapsulation (Getters & Setters)
- Practice Exercises
- Mini Projects

## 🛠️ Technologies

- Dart
- Visual Studio Code
- Git & GitHub

## 🎯 Purpose

This repository is created to document my journey of learning Dart from the basics to object-oriented programming through hands-on coding exercises and projects.

## 🚀 How to Run

1. Install the Dart SDK.
2. Clone the repository:

```bash
git clone https://github.com/AHMED-ali123-ali/Dart-Course.git
```

3. Navigate to the project folder:

```bash
cd Dart-Course
```

4. Run any Dart file:

```bash
dart filename.dart
```
